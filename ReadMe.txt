ReadMe:
Notre projet consiste à la réalisation d'un site web de personnalisation de tshirt avec les notions d'HTML, CSS et JavaScript.
Nous avons fait une page de présentation de notre marque expliquant la raison de notre projet, une page de présentation des 
produits, avec un filtre permettant de mieux selectionner ce qui nous interresse. L'affichage de cette page est gérer 
dynamiquement, c'est à dire qu'il suffit de rajouter un produit avec les bons codes dans le JSON et cela affichera le produit 
ajouté (le nombre de couleur par tshirt est limité à 6 car les ronds de couleurs ne sont pas géré pour géré un affichage dynamique 
joli.
En cliquant sur un produit cela vous amème sur la page de personnalisation, ou vous pouvez choisir la taille, la couleur et la 
quantité pour ensuite l'ajouter au panier. Si vous ajouter le même produit avec deux quantité différentes successivement, les 
quantités s'ajoutent dans le panier. Une réduction de 10% est ajouter si la quantité du produit est supérieure ou égale à 10 
Vous pouvez enlever les produits ou commander. En appuyant sur le bouton commander, cela vous affiche un formulaire à remplir 
avec des champs obligatoires. Le calcul du cout de livraison est ensuite fait:
- 0€ si la livraison est +72h après la commande, 8€ sinon. 
Et un cout de livraison liée a la distance de livraison : 0 si la distance est < 20km, et sinon dépend de la distance.
Vous pouvez enfin valider le panier.

Toutes les images du sites sont libres de droits, ou ont été réaliser par nous.
